# Stuard AI — USP & Green Flags

## Unique Selling Proposition (USP)
Stuard AI: the local-first, trustworthy automation buddy.

- **Local-first workflows:** Run powerful automations and agentic workflows on your machine by default — your data stays local unless you opt into cloud sync.
- **Seamless local + cloud context fusion:** Use context from local files, terminal commands, and desktop apps alongside cloud sources (Outlook, Drive, Docs) in one orchestrated assistant.
- **Translucent overlay UI:** A non-intrusive, translucent pop-up HUD that appears in-context and never captures data silently.
- **Auditable agent actions & undo:** Agents propose actions, show previews, and require consent for high-impact tasks; all runs are logged and reversible.
- **Human-friendly memory:** A safe, editable personal memory that makes the assistant feel like a helpful buddy without compromising privacy.
- **Developer-first extensibility:** Plugin architecture for custom connectors, PowerShell/AutoHotkey hooks, and local LLM integrations (Ollama/LM Studio).

## Green Flags (signals of a trustworthy product)
- **Local-by-default processing** (privacy-preserving). 
- **Granular permissions & OAuth minimization** for cloud integrations. 
- **Explicit consent UI** for any data access (files, clipboard, screen). 
- **Transparent logs & audit trails** for automation and agent decisions. 
- **Editable memory UI** with export/delete options. 
- **Reproducible demos** and clear, evidence-based claims. 
- **No ragebait/hype marketing** — factual, user-centric messaging. 
- **Safety-first defaults**: require confirmation for destructive actions and rate-limit autonomous agents.

## Short positioning line
"Stuard AI — Local-first automation that acts like a reliable teammate: private, auditable, and ready to help."
