# Stuard Agentic Workflows

Vision: A system where users can compose, share, buy, and sell agentic workflows for Stuard. Supports:
- Auto mode: end-to-end agentic execution
- Manual mode: step-by-step with human-in-the-loop
- Marketplace: publishing, rating, billing, and licensing

## Components
- Workflow DSL (YAML/JSON)
- Runtime Orchestrator (state machine)
- Tool Abstraction Layer (capabilities registry)
- Agent Profiles (skills, risk thresholds)
- Human-in-the-loop UI (approvals, edits)
- Marketplace (catalog, payments, reviews)
- Security & Sandbox (scopes, quotas)
- Telemetry (traces, cost, time)
